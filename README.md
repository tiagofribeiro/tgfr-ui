# tgfr-ui

<br>

![logo](/src/logo.svg)

<br>

Biblioteca de componentes pré-configurados.

<br>

### Uso

Desenvolvimento:
```bash
npm link tgfr-ui
# ou
yarn link tgfr-ui
```

Instalação:
```bash
npm install ./tgfr-ui
# ou
yarn add file:./tgfr-ui
```
